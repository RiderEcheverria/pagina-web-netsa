DESARROLLO DE APLICACIONES Y SISTEMAS DE INFORMACIÓN WEB

Diseños Web
El diseño web es un área enfocada en el desarrollo de interfaces digitales, como el diseño de sitios y aplicaciones para web.
Desarrollo web
El desarrollo web es la construcción, creación y mantenimiento de sitios web. Incluye aspectos como diseño web, publicación web, programación web y base de datos gestión.
Diseños responsivos
El diseño responsive, es un tipo de desarrollo de páginas web que permite su óptima visualización en distintos tipos de dispositivos finales.
multiplataforma
Es un tipo de aplicación/programa/software que funciona en varios sistemas operativos o dispositivos, como Windows, Mac OS, Android o iOS.
Bases de datos
Es una recopilación organizada de información o datos estructurados, que normalmente se almacena de forma electrónica en un sistema informático.
aplicaciones
El desarrollo de aplicaciones móviles es el conjunto de procesos y procedimientos involucrados en la escritura de software para teléfonos inteligentes y otros dispositivos portátiles.
